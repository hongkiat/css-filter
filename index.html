CSS3 Image Filter Effects

Webkit introduced a new CSS property that allows us to adjust image through CSS. This demo shows when this property is applied to the image.

grayscale

Photo by: Mehdi Kh

Codes

img {
  -webkit-filter: grayscale(100%);
}
